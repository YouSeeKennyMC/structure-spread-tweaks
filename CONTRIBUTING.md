# Contributing

Contributions are welcome when they keep the mod small, focused, and compatible with Minecraft `26.1.x`.

## Development

1. Clone the repository.
2. Build with `gradle build`.
3. Test with a clean Fabric profile.

## Pull Requests

Please include:

- What changed
- Why it changed
- How it was tested
- Any compatibility impact for modpacks

Avoid unrelated formatting or refactors in behavior PRs.

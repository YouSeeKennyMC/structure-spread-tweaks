# Publishing

## GitHub Repository

Suggested repository name:

```text
structure-spread-tweaks
```

Suggested description:

```text
Configurable Fabric structure spacing for Minecraft 26.1.x modpacks.
```

Suggested topics:

```text
minecraft fabric modding worldgen structures modpack
```

## First Push

After creating an empty GitHub repository, push this local project:

```powershell
git remote add origin https://github.com/YouSeeKennyMC/structure-spread-tweaks.git
git branch -M main
git push -u origin main
```

## Wiki

GitHub Wiki pages are stored in a separate repository. The starter wiki content is in:

```text
docs/WIKI.md
```

After the GitHub repository exists, enable the Wiki in repository settings and copy or import that page as the wiki home page.

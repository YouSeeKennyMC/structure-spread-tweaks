# Changelog

## 1.0.0

- Initial public release.
- Added configurable global structure spread factor.
- Added vanilla and modded structure toggles.
- Added per-structure-set and per-structure custom overrides.
- Added optional id-based structure salt replacement.
- Added Mod Menu and Cloth Config screen.
- Saved in-game changes apply to the next world load or datapack reload without restarting Minecraft.
